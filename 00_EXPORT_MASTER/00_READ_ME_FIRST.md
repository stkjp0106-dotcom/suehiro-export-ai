# SUEHIRO TRADING Export AI Knowledge Base

## Purpose
- Use this folder as the first reference point for export sales, export administration, logistics, and regulatory support.
- Keep information structured as conclusion first, then reasons, then evidence or source file links.
- Do not let AI make final decisions on price, legal requirements, import rules, export eligibility, or food safety/regulatory matters without human confirmation.

## How AI should use this folder
- Read `00_EXPORT_MASTER` first.
- Then check the relevant product, country, factory, logistics, sales template, and case study files.
- If information conflicts, prioritize the newest human-updated file and ask for confirmation.
- If a field says `要入力` or `要確認`, do not guess.

## Human-only confirmation areas
- Price
- Freight quote
- HS code
- Export/import regulation
- Factory approval status
- Labeling rule
- Shelf life acceptability
- Payment terms
- Delivery terms such as FOB, CIF, DAP, DDP
